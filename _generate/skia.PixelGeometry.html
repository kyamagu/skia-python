PixelGeometry¶

class PixelGeometry¶

Description of how the LCD strips are arranged for each pixel.

If this is unknown, or the pixels are meant to be “portable” and/or transformed before showing (e.g. rotated, scaled) then use PixelGeometry.kUnknown.
