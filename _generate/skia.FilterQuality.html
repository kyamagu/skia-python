FilterQuality¶

class FilterQuality¶

Controls how much filtering to be done when scaling/transforming complex colors e.g. image.

Members:

kMedium : typically bilerp + mipmaps for down-scaling

kNone : fastest but lowest quality, typically nearest-neighbor

kHigh : slowest but highest quality, typically bicubic or better

kLow : typically bilerp
